# Backend Server

Express.js server with SQLite database for portfolio message system and admin panel.

## Environment Variables

Create a `.env` file in this directory with:

```env
PORT=3001
JWT_SECRET=your-secret-key
GMAIL_USER=your-email@gmail.com
GMAIL_APP_PASSWORD=your-app-password
```

See `README_SETUP.md` in the root directory for detailed setup instructions.
