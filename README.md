# TechNur Portfolio Hub

A modern, responsive web application built with React, TypeScript, and Tailwind CSS.

## Project Info

**Author**: TechNur
**Description**: A React + Node.js + SQLite project.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
